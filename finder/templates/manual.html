Краткая инструкция для поисковика.

В строке поиска доступны 3 аргумента которые можно писать в любом порядке и с любом регистре ЧЕРЕЗ ПРОБЕЛ.
Что бы найти все возможные варианты при поиске метизов используйте ОБЯЗАТЕЛЬНО в качестве разделителя '*': Пример 4*12

Четвертый аргумент в строке поиска - это комментарий. В результате отобразится точное совпадение, если у позиции есть коментарий.

Для поиска АРТИКУЛА можно использовать краткую запись состоящую только из цифр, но для более точного совпадения рекомендуется вводить атрикул полностью.

Для поиска КОДА используйте вначале симвал '*'

Для поиска ТОЛЬКО ПО КОММЕНТАРИЮ используйте вначале симвал '-'.
Так же доступны четыре аргумента которые можно писать в любом порядке и в любом регистре ЧЕРЕЗ ПРОБЕЛ!

Выбор проекта для сортировки.
В левом столбце выберите вкладку "Выбрать проект"
Выделите нужные вам позиции, нажмите "Применить фильтр".
Поисковые запросы будут фильтроваться относительно выбранных проектов.
Что бы убрать сортировку нажмите вкладку "Очистить фильтр"
