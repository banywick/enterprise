Вернуться к поиску

Краткая инструкция для поисковика.

Поиск:
В строке поиска доступны 4 аргумента :
(Название метиза) (5*6) (Гост,Дин) (А4,А2) которые можно писать в любом порядке и с любым регистром. через пробел.
A4 И А2 Пишут с русской буквой
Чтобы найти все возможные варианты при поиске метизов, используйте обязательно в качестве разделителя '*': Пример 4*12.
Примечание: не обязательно писать все аргументы, но для более точного совпадения по поиску рекомендуется использовать всё.


Поиск и комментарий
Пятый аргумент в строке поиска - это комментарий. (не обязательный) В результате отобразится точное совпадение, если у позиции есть комментарий.


Поиск Артикула
Для поиска артикула можно использовать краткую запись, состоящую только из цифр, но для более точного совпадения рекомендуется вводить артикул полностью.


Поиск Кода
Для поиска кода используйте вначале символ *.


Поиск Комментария
Для поиска ТОЛЬКО ПО КОММЕНТАРИЮ используйте вначале символ -.


Сортировка
Для сортировки по проектам нажмите в левой колонке
"Выберать проект". На предложенной странице выберите нужные для сорировки объекты и нажмите "Применить фильтр". В левом столбце вы увидите выбранные элементы по которым будет проводится сортировка. Что бы убрать сортировку нажмите в левой колонке "Очистить фильтр"
